Support Desk

Visit Support Desk View demo View code

Overview

Problem: A big company welcome each new employee with: 1 laptop, 1 desktop screen, 1 keyboard, 1 mobile phone, 1 tablet, 1 badge for the employee gate, 1 badge for the cafeteria, 1 Welcome ebook. Technicians need a platform to manage their daily support for all this equipment.

Solution: a website with 2 modules

  • for employees who can send tickets, receive and answer notes, manage their own profile
  • for IT administrators who can manage products, receive tickets, send notes and manage employees profile

What I learned

  • Remix Web framework (compiler, server-side HTTP handler, server framework, browser framework) + React 18 + React Router 6 + Typescript
  • Prisma (Next-generation Node.js and TypeScript Object Relational Mapping) + MongoDB
  • Deployment on Vercel

ShopLineArt

Overview

Problem: A blogger needs to illustrate her posts. She likes to navigate to Pexels.com which is a free stock photos of amazing artists. She would like to spend less time selecting and downloading pictures.

Solution: a website with 3 main features

  • photos search by term,
  • display of the photos in different sizes,
  • protected multiple photos download, zip folder format with credits availability

What I learned

  • React + React-router-dom V6
  • React Context
  • Auth route vs public routes + MongoDB / Mongoose
  • Pexels API Photo Client
  • Express
  • Dark / Light Theme CSS
  • EmailJS for contact form
  • JSzip + file-saver to add custom text files to photo download
  • Server linked to Client Deployments on Render.com
